# Uiverse.io Component Integration

All components have been updated to use uiverse.io styled elements. The following styles are available:

## Available Uiverse.io Classes

### Buttons
- `.btn-uiverse` - Main button style with gradient and hover effects
- `.ripple-uiverse` - Ripple effect on click
- Use `UiverseButton` component for consistent styling

### Inputs
- `.input-uiverse` - Modern input with glassmorphism
- Use `UiverseInput` component for consistent styling

### Cards
- `.card-uiverse` - Glassmorphism card with hover effects
- Use `UiverseCard` component for consistent styling

### Icon Buttons
- `.icon-btn-uiverse` - Circular icon button with animations
- Use `UiverseIconButton` component for consistent styling

### Other Elements
- `.badge-uiverse` - Gradient badge
- `.spinner-uiverse` - Loading spinner
- `.toggle-uiverse` - Toggle switch
- `.gradient-text-uiverse` - Animated gradient text
- `.glow-uiverse` - Glowing effect
- `.pulse-uiverse` - Pulse animation
